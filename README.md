# The Embroidery Atelier 🧵✨

A premium, fully responsive e-commerce platform dedicated to the art of handcrafted embroidery. This project showcases a blend of traditional craftsmanship and modern web technology, designed for elegance and high conversion.

## 🌟 Key Features

- **Dynamic E-commerce Flow:** A seamless shopping experience with functional "Add to Cart" and "Buy Now" capabilities.
- **Persistent Shopping Cart:** Built with React Context and LocalStorage to ensure user data persists across sessions and refreshes.
- **Secure Checkout System:** A custom-built checkout flow that captures customer details and order summaries, powered by FormSubmit for secure order notifications.
- **Modern Animations:** Integrated with **AOS (Animate On Scroll)** and **Framer Motion** for a smooth, "alive" user interface.
- **Responsive Design:** Fully optimized for all screen sizes, from mobile devices to large desktops.
- **Brand Consistency:** Personalized branding throughout, reflecting a sophisticated and artistic aesthetic.

## 🛠️ Technical Stack

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** Custom CSS with Tailwind CSS foundation
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
- **Animations:** [AOS](https://michalsnik.github.io/aos/) & [Framer Motion](https://www.framer.com/motion/)
- **Backend/Forms:** [FormSubmit](https://formsubmit.co/)

## 🚀 Getting Started

First, clone the repository and install the dependencies:

```bash
git clone https://github.com/Fatimahnoman/The-Embroidery-Atelier.git
cd The-Embroidery-Atelier
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
